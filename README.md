# Nokia Snake Game

A browser-based Snake game with a Nokia-inspired aesthetic.

## How to Play

- **Desktop:** Use the arrow keys to control the snake
- **Mobile:** Swipe on the game canvas to change direction

Press any arrow key (or swipe) to start. Do the same to restart after a game over.

## Features

- Classic snake gameplay on a 20×20 grid
- Nokia-style font and color palette
- Score tracking with persistent high score (saved in localStorage)
- Darker head color so you can always tell which way the snake is facing
- Works on both desktop and mobile
